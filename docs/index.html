mksql (github)

a tool to make & query sqlite databases

ETL intended for people who:

  • like SQL as their daily driver
  • have modest data needs (< ~64 TB, < ~100 qps)

I previously posted a side project squirrelbyte (Hacker News) ... this project has similar motivations ... let's see how it goes

Add your email to the google form here or the window at the bottom of the page to get email updates about the project

the gist

1) put data in

upload files like csv, tsv, json, xls, sqlite, etc

any size that fits on disk is fine: 5 KB, 5 MB, 5 GB, 5 TB, .... AWS EBS supports up to 64 TB

run a local cli or hit a remote server

use a command like:

http --form POST https://mksql.com/u file@data.csv

2) get data out ... with SQL

run arbitrary sql queries against your data

get the data back in the shape you want it: json, tsv, <table>, <div>, etc

invoke in static websites, jupyter notebooks, observable hq, etc.

use a command like:

echo "SELECT name, favorite_color FROM person WHERE favorite_color = '#fcd703'" | http POST https://mksql.com/q
